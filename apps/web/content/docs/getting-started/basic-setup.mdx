## Config Paths

Default config location: `~/.config/mango/config.conf`

Autostart script: `~/.config/mango/autostart.sh`

Copy default config:

```bash
cp /etc/mango/config.conf ~/.config/mango/config.conf
touch ~/.config/mango/autostart.sh
chmod +x ~/.config/mango/autostart.sh
```

Custom path via environment: `MANGOCONFIG=/path/to/config`

## Basic Configuration

Create `~/.config/mango/config.conf`:

```ini
# Terminal and launcher
bind=ALT,Return,spawn,foot
bind=ALT,Space,spawn,rofi -show drun

# Window management
bind=ALT,q,killclient
bind=ALT,Left,focusdir,left
bind=ALT,Right,focusdir,right
bind=ALT,Up,focusdir,up
bind=ALT,Down,focusdir,down

# Quit
bind=SUPER,m,quit
```

<Callout type="info">
  Bindings use keycode by default. Use `bind[flag]` for keysym or release events.
</Callout>

## Autostart Script

Edit `~/.config/mango/autostart.sh`:

```bash
#!/bin/bash

# Wallpaper
swaybg -i /path/to/wallpaper.png &

# Status bar
waybar &

# Notifications
swaync &

# Clipboard
wl-paste --type text --watch cliphist store &
```

<Callout type="warn">
  Make the script executable with `chmod +x ~/.config/mango/autostart.sh`
</Callout>

## Environment Variables

Common environment setup:

```ini
# IME support
env=GTK_IM_MODULE,fcitx
env=QT_IM_MODULE,fcitx
env=XMODIFIERS,@im=fcitx

# Qt scaling
env=QT_AUTO_SCREEN_SCALE_FACTOR,1
env=QT_QPA_PLATFORM,Wayland
```
