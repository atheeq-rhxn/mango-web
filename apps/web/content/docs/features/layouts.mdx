## Supported Layouts

MangoWC supports multiple window layout modes:

- tile
- scroller
- monocle
- grid
- deck
- center_tile
- vertical_tile
- vertical_grid
- vertical_scroller
- vertical_spiral
- vertical_deck

## Switching Layouts

Bind layout switch:

```ini
bind=SUPER,l,setlayout,scroller
bind=SUPER+SHIFT,l,switch_layout
```

IPC command:

```bash
mmsg -l "scroller"
```

## Layout Commands

| Command | Parameters | Description |
|---------|------------|-------------|
| `setlayout` | layout_name | Set specific layout |
| `switch_layout` | - | Cycle through layouts |
| `incnmaster` | +1/-1 | Adjust master window count |
| `setmfact` | +/-value | Adjust master area factor |
| `zoom` | - | Swap with master window |

## Master-Stack Settings

Applies to tile, center_tile, etc.

| Setting | Default | Description |
|---------|---------|-------------|
| `new_is_master` | 1 | New windows become master |
| `default_mfact` | 0.55 | Master area ratio (0.1-0.9) |
| `default_nmaster` | 1 | Number of master windows |
| `smartgaps` | 0 | Smart gaps when single window |
| `center_master_overspread` | 0 | Master overspread in center_tile |
| `center_when_single_stack` | 1 | Center master with one stack |

## Scroller Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `scroller_structs` | 20 | Side padding |
| `scroller_default_proportion` | 0.8 | Default width ratio |
| `scroller_focus_center` | 0 | Always center focused |
| `scroller_prefer_center` | 0 | Prefer center if visible |
| `scroller_ignore_proportion_single` | 0 | Ignore ratio for single |
| `scroller_default_proportion_single` | 1.0 | Single window ratio |
| `scroller_proportion_preset` | 0.5,0.8,1.0 | Ratio presets |

Scroller commands:

| Command | Description |
|---------|-------------|
| `set_proportion` | Set width ratio (0.0-1.0) |
| `switch_proportion_preset` | Cycle ratio presets |

## Tag-Specific Layouts

Set layout per tag:

```ini
tagrule=id:1,layout_name:scroller
tagrule=id:2,layout_name:center_tile
```

## Layout Indicators

Waybar integration shows current layout.

Config example:

```json
"dwl/window": {
  "format": "[{layout}]{title}"
}
```
