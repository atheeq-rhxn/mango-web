## Overview

Control MangoWC externally using the `mmsg` command.

## Basic Syntax

```bash
mmsg [-OTL]
mmsg [-o <output>] -s [-t <tags>] [-l <layout>] [-c <tags>]
mmsg [-o <output>] (-g | -w) [-Ootlcvmf] [-d <cmd>,<arg1>,<arg2>]
```

## Options

| Option | Description |
|--------|-------------|
| `-q` | Quit MangoWC |
| `-g` | Get value |
| `-s` | Set value |
| `-w` | Watch for changes |
| `-O` | Get all outputs |
| `-T` | Get number of tags |
| `-L` | Get all layouts |
| `-o` | Select output |
| `-t` | Get/set selected tags |
| `-l` | Get/set layout |
| `-c` | Get focused client info |
| `-v` | Get statusbar visibility |
| `-m` | Get fullscreen status |
| `-f` | Get floating status |
| `-d` | Execute command |
| `-x` | Get focused geometry |
| `-e` | Get last layer name |
| `-k` | Get keyboard layout |
| `-b` | Get keymode |
| `-A` | Get monitor scale |

## Execute Commands

```bash
mmsg -d killclient
mmsg -d spawn,foot
mmsg -d resizewin,+10,+10
```

## Switch Layout

Supported layouts:

- "S" → scroller
- "T" → tile
- "G" → grid
- "M" → monocle
- "K" → deck
- "VS" → vertical scroller
- "VT" → vertical tile
- "CT" → center tile

```bash
mmsg -l "S"
```

## Switch Tag

```bash
mmsg -t 1  # Switch to tag 1
mmsg -t 2+ # Add to current view
mmsg -t 2- # Remove from view
mmsg -t 2^ # Toggle in view
```

## Get Information

```bash
mmsg -w        # Watch all changes
mmsg -g        # Get all info
mmsg -w -c     # Watch focused client
mmsg -O        # Get all outputs
mmsg -g -t     # Get tag info
mmsg -g -c     # Get client info
```

## Tag Information

Output format:

```
Monitor | Tag | State | Clients | Focused
eDP-1   | 2   | 0     | 1       | 0
```

State: 0=none, 1=active, 2=urgent

Mask format:

```
Monitor | Occupied | Active | Urgent
eDP-1   | 14       | 6      | 0
```

## Scripting Examples

Check if window is focused:

```bash
if mmsg -g -f | grep -q "1"; then
  echo "Window is floating"
fi
```

Get current layout:

```bash
layout=$(mmsg -g -l)
echo "Current layout: $layout"
```

Watch for tag changes:

```bash
mmsg -w -t | while read line; do
  echo "Tag changed: $line"
done
```
