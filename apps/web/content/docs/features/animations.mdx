## Animation Overview

MangoWC provides smooth animations for window operations and layer changes.

Layer animation: https://github.com/user-attachments/assets/014c893f-115c-4ae9-8342-f9ae3e9a0df0

## Animation Settings

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `animations` | int | 1 | Enable window animations |
| `layer_animations` | int | 0 | Enable layer animations |
| `animation_type_open` | string | slide | Open animation type |
| `animation_type_close` | string | slide | Close animation type |
| `layer_animation_type_open` | string | slide | Layer open type |
| `layer_animation_type_close` | string | slide | Layer close type |
| `animation_fade_in` | int | 1 | Enable fade-in |
| `animation_fade_out` | int | 1 | Enable fade-out |
| `tag_animation_direction` | int | 1 | Tag direction (1=horizontal, 0=vertical) |
| `zoom_initial_ratio` | float | 0.3 | Zoom start ratio |
| `zoom_end_ratio` | float | 0.8 | Zoom end ratio |
| `fadein_begin_opacity` | float | 0.5 | Fade-in start opacity |
| `fadeout_begin_opacity` | float | 0.8 | Fade-out start opacity |

## Animation Types

- **zoom**: Windows scale in/out
- **slide**: Windows slide in direction
- **fade**: Opacity transition
- **none**: No animation

## Duration Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `animation_duration_move` | 500 | Move duration (ms) |
| `animation_duration_open` | 400 | Open duration (ms) |
| `animation_duration_tag` | 350 | Tag switch duration (ms) |
| `animation_duration_close` | 800 | Close duration (ms) |
| `animation_duration_focus` | 0 | Focus opacity duration (ms) |

## Animation Curves

Bezier curves for smooth transitions.

Format: `x1,y1,x2,y2`

| Setting | Default | Description |
|---------|---------|-------------|
| `animation_curve_open` | 0.46,1.0,0.29,1 | Open curve |
| `animation_curve_move` | 0.46,1.0,0.29,1 | Move curve |
| `animation_curve_tag` | 0.46,1.0,0.29,1 | Tag curve |
| `animation_curve_close` | 0.08,0.92,0,1 | Close curve |
| `animation_curve_focus` | 0.46,1.0,0.29,1 | Focus curve |

## Custom Curves

Design curves at [cssportal.com/cubic-bezier-generator](https://www.cssportal.com/css-cubic-bezier-generator) or [easings.net](https://easings.net).

Example custom curves:

```ini
animation_curve_open=0.46,1.0,0.29,1.1
animation_curve_close=0.08,0.92,0,1
```

## Performance Notes

- Blur effects impact performance
- `blur_optimized=1` caches backgrounds
- Disable animations on low-end hardware
- Layer animations are optional

## Examples

Window open animation:

```ini
animation_type_open=zoom
zoom_initial_ratio=0.3
animation_duration_open=400
animation_curve_open=0.46,1.0,0.29,1
```

Tag switch animation:

```ini
tag_animation_direction=1
animation_duration_tag=350
animation_curve_tag=0.46,1.0,0.29,1
```
