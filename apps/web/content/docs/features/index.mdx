## Key Capabilities

- **Dynamic Layouts**: Multiple layout modes with per-tag settings
- **Smooth Animations**: Customizable transitions for all window operations
- **IPC Control**: Full control via external programs
- **Modern Integrations**: Seamless desktop environment integration
- **Advanced Features**: Scratchpad, virtual monitors, tearing support

## Layouts

Supported layouts:

- tile, scroller, monocle, grid
- deck, center_tile, vertical variants
- Customizable master-stack and scroller settings

## Animations

- Window open/close/move/tag animations
- Layer animations
- Customizable curves and durations
- Fade effects and zoom ratios

## IPC

- Control compositor from scripts
- Get window and tag information
- Watch for changes in real-time
- Switch layouts, views, and execute commands

## Integrations

- Waybar configuration and modules
- Desktop portals for file picker, clipboard
- Screen recording and sharing
- Notification daemons and status bars
