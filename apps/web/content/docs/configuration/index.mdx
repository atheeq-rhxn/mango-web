## Configuration Overview

MangoWC uses external configuration files for all settings. Changes reload automatically with `reload_config` command.

<Callout>
  Configuration is hot-reloadable - no restart required for most changes.
</Callout>

## Config Structure

Configuration uses key-value pairs:

```ini
# Comments start with #
setting_name=value

# Commands
bind=modifiers,key,command,parameters

# Rules
windowrule=parameter:value,appid:name
```

## Reloading Config

Reload configuration without restarting:

```bash
bind=SUPER,r,reload_config
```

Or via IPC:

```bash
mmsg -d reload_config
```
