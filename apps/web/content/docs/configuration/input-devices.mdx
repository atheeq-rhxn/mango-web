## Keyboard Settings

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `repeat_rate` | integer | 25 | Keyboard repeat rate |
| `repeat_delay` | integer | 600 | Keyboard repeat delay |
| `numlockon` | integer | 1 | Numlock state on startup |

## Keyboard layout and IME

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `xkb_rules_rules` | string | - | XKB keyboard mapping rules |
| `xkb_rules_model` | string | - | Keyboard hardware model |
| `xkb_rules_layout` | string | us | Country/region code |
| `xkb_rules_variant` | string | - | Layout variants |
| `xkb_rules_options` | string | - | Additional options |

Example:

```ini
xkb_rules_layout=us,us
xkb_rules_variant=dvorak,
xkb_rules_options=grp:lalt_lshift_toggle
```

### IME env for fcitx5

```ini
env=GTK_IM_MODULE,fcitx
env=QT_IM_MODULE,fcitx
env=SDL_IM_MODULE,fcitx
env=XMODIFIERS,@im=fcitx
env=GLFW_IM_MODULE,ibus
```

## Trackpad Settings & Mouse Settings

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `disable_trackpad` | integer | 0 | Disable trackpad |
| `tap_to_click` | integer | 1 | Enable tap to click |
| `tap_and_drag` | integer | 1 | Enable tap and drag |
| `drag_lock` | integer | 1 | Enable drag lock |
| `trackpad_natural_scrolling` | integer | 0 | Natural scrolling direction |
| `disable_while_typing` | integer | 1 | Disable trackpad while typing |
| `left_handed` | integer | 0 | Left-handed mode |
| `middle_button_emulation` | integer | 0 | Middle mouse button emulation |
| `swipe_min_threshold` | integer | 20 | Minimum swipe threshold |
| `mouse_natural_scrolling` | integer | 0 | Natural scrolling for mouse |
| `accel_profile` | integer | 2 | Accel mode |
| `accel_speed` | float | 0.0 | Accel speed |
| `scroll_method` | integer | 1 | Scrolling method for touchpads |
| `click_method` | integer | 1 | Method for triggering button clicks |
| `send_events_mode` | integer | 0 | Controls when events are sent |
| `button_map` | integer | 0 | Button mapping |

**Detailed Descriptions:**

1. **scroll_method**
   - `0`: Never send scroll events instead of pointer motion events.
   - `1`: Send scroll events when two fingers are logically down on the device.
   - `2`: Send scroll events when a finger moves along the bottom or right edge of a device.
   - `4`: Send scroll events when a button is down and the device moves along a scroll-capable axis.

2. **click_method**
   - `0`: No software click emulation.
   - `1`: Use software-button areas to generate button events.
   - `2`: The number of fingers decides which button press to generate.

3. **send_events_mode**
   - `0`: Send events from this device normally.
   - `1`: Do not send events through this device.
   - `2`: If an external pointer device is plugged in, do not send events from this device.

4. **button_map**
   - `0`: left/right/middle
   - `1`: left/middle/right

5. **accel_profile**
   - `0`: Don't use accel
   - `1`: No dynamic acceleration. Pointer speed = original input speed × (1 + accel_speed).
   - `2`: Slow movement leads to less acceleration, while fast movement leads to more acceleration.

## Key Bindings

### Format

`bind=<MODIFIERS>,<KEY>,<COMMAND>,<PARAMETERS>`

- `<MODIFIERS>`: Comma-separated modifier keys (SUPER,CTRL,ALT,SHIFT,NONE), combine with `+`
- `<KEY>`: Keyname from xev output
- `<COMMAND>`: Action to execute
- `<PARAMETERS>`: Optional parameters

### Bind flag

`bind[flag]`, flag can be:
- `l`: still apply bind when screen is locked
- `s`: use keysym instead of keycode
- `r`: handle release event instead of press

```ini
bindl=SUPER,l,quit
binds=SUPER,s,quit
```

### Key mode

Group keys into modes.

- Set `keymode=name` before binds
- `common` mode applies to all
- `default` is the base mode
- Use `setkeymode` to switch

## Mouse Bindings

Format: `mousebind=modifiers,button,command,parameters`

Buttons: btn_left, btn_right, btn_middle

Examples:

```ini
mousebind=SUPER,btn_left,moveresize,curmove
mousebind=SUPER,btn_right,moveresize,curresize
```

## Axis Bindings

Format: `axisbind=modifiers,direction,command,parameters`

Directions: UP, DOWN, LEFT, RIGHT

Examples:

```ini
axisbind=SUPER,UP,viewtoleft_have_client
axisbind=SUPER,DOWN,viewtoright_have_client
```

## Gesture Bindings

Touchpad gesture binds.

`gesturebind=<MODIFIERS>,<DIRECTION>,<FINGERS>,<COMMAND>,<PARAMETERS>`

- `<MODIFIERS>`: Comma-separated modifier keys
- `<DIRECTION>`: up/down/left/right
- `<FINGERS>`: Number of fingers (3,4)

Examples:

```ini
gesturebind=none,left,3,focusdir,left
gesturebind=none,right,3,focusdir,right
gesturebind=none,up,3,focusdir,up
gesturebind=none,down,3,focusdir,down
gesturebind=none,left,4,viewtoleft_have_client
gesturebind=none,right,4,viewtoright_have_client
gesturebind=none,up,4,toggleoverview
gesturebind=none,down,4,toggleoverview
```

## Switch Bindings

Binds for lid fold toggle.

Disable system lid handle in `/etc/systemd/logind.conf`:

```ini
HandleLidSwitch=ignore
HandleLidSwitchExternalPower=ignore
HandleLidSwitchDocked=ignore
```

`switchbind=<FOLD_STATE>,<COMMAND>,<PARAMETERS>`

- `<FOLD_STATE>`: fold/unfold

Examples:

```ini
switchbind=fold,spawn,swaylock -f -c 000000
switchbind=unfold,spawn,wlr-dpms on
```
