## Window Rules

Apply settings to specific windows. Match by appid, title, or both.

Format: `windowrule=parameter:value,condition:value`

| Parameter | Values | Description |
|-----------|--------|-------------|
| `appid` | string | Match application ID (regex) |
| `title` | string | Match window title (regex) |
| `tags` | 1-9 | Assign to tag |
| `isfloating` | 0/1 | Force floating state |
| `isfullscreen` | 0/1 | Force fullscreen |
| `scroller_proportion` | 0.1-1.0 | Set scroller width ratio |
| `animation_type_open` | zoom/slide/fade/none | Open animation |
| `animation_type_close` | zoom/slide/fade/none | Close animation |
| `isnoborder` | 0/1 | Remove border |
| `monitor` | name | Assign to monitor |
| `offsetx/offsety` | -100 to 100 | Position offset (%) |
| `width/height` | pixels | Floating size |
| `isglobal` | 0/1 | Global window |
| `isoverlay` | 0/1 | Always on top |
| `force_tile_state` | 0/1 | Force tiled |

Examples:

```ini
windowrule=isfloating:1,title:Calculator
windowrule=tags:2,appid:firefox
windowrule=width:1000,height:800,appid:yesplaymusic
```

## Tag Rules

Configure tags per monitor.

Format: `tagrule=id:value,parameter:value`

| Parameter | Values | Description |
|-----------|--------|-------------|
| `id` | 1-9 | Tag number (0 = all tags) |
| `monitor_name` | string | Monitor name |
| `layout_name` | layout | Default layout |
| `no_render_border` | 0/1 | Disable borders |
| `no_hide` | 0/1 | Keep tag visible |

Examples:

```ini
tagrule=id:1,no_hide:1
tagrule=id:2,monitor_name:eDP-1,no_hide:1
tagrule=id:3,layout_name:scroller
```

## Layout Settings

### Master-Stack

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `new_is_master` | int | 1 | New windows become master |
| `default_mfact` | float | 0.55 | Master area ratio |
| `default_nmaster` | int | 1 | Number of master windows |
| `smartgaps` | int | 0 | Enable smart gaps |
| `center_master_overspread` | int | 0 | Center master overspread (center_tile) |
| `center_when_single_stack` | int | 1 | Center with single stack (center_tile) |

### Scroller

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `scroller_structs` | int | 20 | Side padding |
| `scroller_default_proportion` | float | 0.8 | Default width ratio |
| `scroller_focus_center` | int | 0 | Always center focused |
| `scroller_prefer_center` | int | 0 | Prefer center if visible |
| `scroller_ignore_proportion_single` | int | 0 | Ignore ratio for single window |
| `scroller_default_proportion_single` | float | 1.0 | Single window ratio |
| `scroller_proportion_preset` | string | 0.5,0.8,1.0 | Ratio presets |

### Overview

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `hotarea_size` | int | 10 | Hot area size (pixels) |
| `enable_hotarea` | int | 1 | Enable hot areas |
| `ov_tab_mode` | int | 0 | Tab mode |
| `overviewgappi` | int | 5 | Inner gap |
| `overviewgappo` | int | 30 | Outer gap |

<Callout>
  Hot area: Cursor in bottom-left corner toggles overview. Right-click closes windows, left-click focuses.
</Callout>
