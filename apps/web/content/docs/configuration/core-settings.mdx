## Config Paths

### Using the default config path

- The default config path is `~/.config/mango/config.conf`

- The fallback config path is in `/etc/mango/config.conf`, you can find the default config here

```sh
cp /etc/mango/config.conf ~/.config/mango/config.conf
```

### Sub config

You can use `source` keyword to include other config file.

```ini
# absolute path
source=~/.config/mango/bind.conf
# relative path
source=./bind.conf
```

### Custom config path

```bash
mango -c your_config_file_path
```

## Run app in startup

```ini
# only run once when start mango
exec-once=swaybg -i xxx.png
# run every time after reload config
exec=bash reloadsomething.sh
```

## Env setting

```ini
env=<ENV NAME>,<ENV VALUE>
```

For example:

```ini
env=HELLO,word
```

Common examples:

```ini
# IME
env=GTK_IM_MODULE,fcitx
env=QT_IM_MODULE,fcitx
env=XMODIFIERS,@im=fcitx

# Qt
env=QT_AUTO_SCREEN_SCALE_FACTOR,1
env=QT_QPA_PLATFORM,Wayland

# Cursor
env=XCURSOR_SIZE,24
```

## Reload Config Bind

```ini
bind=SUPER,r,reload_config
```

## Basic Settings

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `xwayland_persistence` | int | 1 | Enable xwayland persistence |
| `syncobj_enable` | int | 0 | Enable drm_syncobj |
| `adaptive_sync` | int | 0 | Enable variable refresh rate |
| `allow_shortcuts_inhibit` | int | 1 | Allow shortcut inhibition |
| `focus_on_activate` | int | 1 | Focus on window activate |
| `sloppyfocus` | int | 1 | Focus follows mouse |
| `warpcursor` | int | 1 | Warp cursor on focus change |

## Monitor Settings

Configure monitors:

```ini
monitorrule=name,mfact,nmaster,layout,transform,scale,x,y,width,height,rate
monitorrule=eDP-1,0.55,1,tile,0,1,0,0,1920,1080,60
```

Transform values: 0=normal, 1=90°, 2=180°, 3=270°, 4=flip vertical, etc.
